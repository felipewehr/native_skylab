# native-skylab
Mobile application on React Native, a project created in a Rocketseat course.


## Global dependencies
- React Native Cli
- Java-jdk8
- Python2
- NodeJs
- Android Debug Bridge (adb)
- Yarn

## Initialize project
1) Add the dependencies.
2) `react-native run-android` or `react-native run-ios`.
